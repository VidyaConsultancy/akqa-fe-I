# TODOAPP

## WHY
- Schedule tasks
- Remember tasks
- Organise the day
- Organise the time

## HOW
- Technology

## WHAT
**MVP - Minimal Viable Product**
- [ ] Add a todo
- [ ] Modifiy a todo
  - [ ] Toggle todo status complete/incomplete
- [ ] Read a todo
- [ ] Read todos list
- [ ] Delete a todo

**Nice to have features**
- [ ] Todo limit warning for a day
- [ ] Rewards point on task/todo completion
- [ ] Reminder/alert

## DATA MODELING
**TODO MODEL**
```json
{
  "id": 1,
  "todo": "Learn React js",
  "isCompleted": false,
  "createdAt": "2022-09-16 11:11:11Z",
  "modifiedAt": "2022-09-16 11:11:11Z",
}
```
